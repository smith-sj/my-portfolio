Skills

I'm finishing up a 10-month bootcamp in fullstack web development at Coder Academy, Melbourne. Working towards a Diploma of Information Technology, while learning HTML, CSS, Ruby, Rails, PostrgeSQL, JavaScript and the MERN Stack (MongoDB, Express, React, Node).

HTML & CSS

Focusing on mobile-first responsive layouts, accessible design and semantic elements, I've learned to craft flexible user-friendly websites. I'm passionate about building with custom CSS, and have experience with various CSS libraries as well.

Ruby & Ruby on Rails

In Ruby I explored the foundational concepts of programming, as well as principles such as OOP, DRY and TDD. Rails was my first exposure to a web framework, as well as the MVC architecture. Through Rails I also learned about relational database management, CRUD operations, SQL queries and user authentication.

JavaScript

Before diving into JS frameworks, I learned vanilla JavaScript, as well as the fundamentals of DOM manipulation and The Event Loop. Through JavaScript I also explored asyncronous programming and browser / third-party APIs.

MERN (Mongo, Express, React, Node)

Now in the final stage of my course, I am learning the MERN Stack, with the goal of creating a full-stack application.
